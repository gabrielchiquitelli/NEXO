# Vista Nova

Site estático de loja de roupa com catálogo, filtros, busca, favoritos, carrinho persistente e checkout simulado.

## Como abrir

Abra `index.html` no navegador ou rode um servidor local na pasta do projeto:

```powershell
python -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Arquivos

- `index.html`: estrutura da loja.
- `styles.css`: layout responsivo e visual.
- `app.js`: catálogo, filtros, favoritos, carrinho e checkout.
